Сервер:
1. Добавить элемент в Commands
2. Добавить реализацию команды в Functions
3. Добавить элемент в comands_init()

Структура команды с сервера:
номер команды / номер команды + '&' + текст

Клиент:
1. Добавить элемент в Commands
2. Добавить реализацию команды в Functions
3. Добавить элемент в comands_init()

Для отпракви сообщения с сервера любого содержания используется (int)Com::message + '&' + std::string message
